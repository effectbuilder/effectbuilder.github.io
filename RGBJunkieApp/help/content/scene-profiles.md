# What is a Scene?

A **Scene** saves everything about your current workspace together:

* Device layout and components
* Canvas tabs and their order
* Effect choices and slider values

Use Scenes when you want different looks for gaming, work, or streaming without rebuilding from scratch.

## Save a Scene

1. Set up your layout and effects the way you want them.
2. In the bar **above the workspace canvas tabs**, click **Save**.
3. Enter a name when prompted — even if a scene is already selected, Save always asks so you can overwrite or save a copy.

Scene files are stored under `profiles/scenes/` in your RGBJunkie AppData folder.

## Load a Scene

Use the **Scene picker** in the same bar above the canvas tabs. Choosing a scene restores the saved layout and effect settings.

> **Tip:** The in-app guide under **Settings → Help** includes keyboard shortcuts and more Scene tips in your language.
