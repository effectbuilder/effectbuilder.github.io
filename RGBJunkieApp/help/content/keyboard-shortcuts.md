## Faster than clicking everything

These shortcuts speed up layout work once your hands know them. They’re ignored while you’re typing in a text field, dropdown, or the help editor.

On macOS, use **Cmd** where **Ctrl** is listed below.

## Workspace

| Shortcut | Action |
| -------- | ------ |
| **Ctrl+0** | Zoom workspace to fit the panel |
| **Ctrl** + wheel | Zoom in/out (pointer over canvas) |
| **W A S D** | Pan when zoomed in |
| **↑ ↓ ← →** | Nudge selection (grid step; **Shift** = 4px; **Ctrl** = 1px) |
| **C** | Center selection on canvas |
| **L** | Lock / unlock selected components |
| **H** | Hide selected components on canvas |
| **Ctrl+G** | Group selection (2+ components) |
| **Ctrl+Shift+G** | Ungroup |
| **1**–**9** | Move selection to canvas tab 1–9 |

## Selection

| Shortcut | Action |
| -------- | ------ |
| **Ctrl+A** | Select all components visible on the active tab |
| **Escape** | Clear canvas selection (when no modal is open) |

## Overlays

| Shortcut | Action |
| -------- | ------ |
| **Escape** | Close whatever’s on top: Settings, Browse effects, Component Library, RGB Wizard, plugin settings, confirm dialogs; exit maximized canvas |

## Diagnostics

| Shortcut | Action |
| -------- | ------ |
| **Ctrl+Shift+F** | Memory tidy (tray / hidden window tuning) |
| **Ctrl+Shift+M** | Save memory snapshot for support |

The full list (with updates) also lives in the app: **Settings → Help → Keyboard shortcuts**.
