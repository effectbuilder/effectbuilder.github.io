## Keep lights running in the background

Closing the window doesn’t have to mean quitting. RGBJunkie can live in the **system tray** so effects and device links stay up.

* **Settings → System → Minimize to tray** — close button hides the window instead of exiting
* **Settings → System → Start minimized** — boot straight to the tray (pairs well with startup)

Right-click the tray icon for **Show**, **Pause effects**, **Check for updates**, and **Quit**.

## Start with Windows

**Settings → System → Start with Windows** adds RGBJunkie to login (portable installs might differ — read the note on that page).

Want lights on as soon as you log in? Turn on **Start with Windows** and **Minimize to tray**, and leave it running.

## Actually quit

**Tray → Quit** (or exit from the app menu) stops effects and lets go of USB devices.

If minimize-to-tray is on, clicking **X** on the window is **not** quitting — the tray icon is still there, still working.
