Component catalog (JSON)

This page uses the same in-browser Firebase connection as the Component Builder (no Node, no npm, no Cloud Functions). It loads public components from the cloud, then builds a JSON file with names, brand, type, creator, stats, image URL when it is a normal link, and rgbjunkie.com URLs for the builder and the direct SignalRGB JSON link (?id=…&export=rgbjunkie).

Click “Build catalog” to load from Firestore.

Optional URL switches: ?autostart=1 — build on load; ?download=1 — after build, download automatically (combine with autostart); ?max=3000; ?includeDataImages=1.

Links use https://rgbjunkie.com. Firestore rules must allow reads (same as the public gallery).
