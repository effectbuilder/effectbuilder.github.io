## Where the fun lives

The **Effects** panel on the right is a **collapsible dock** — use the chevron on the panel edge to hide it for more canvas room. Inside you’ll find:

* **Master brightness** — scales all output before it hits hardware
* **Browse effects** — the full catalog
* **Transport** — shuffle, previous, next, pause/play, and **Auto** cycle
* **Auto-cycle every** — how long each effect stays active when Auto is playing (remembered **per canvas tab**)
* **Effect settings** — sliders and toggles for the active effect appear below when an effect is running

The **Scene** picker lives in the bar **above the workspace canvas**, not in this column. Scenes remember which effect and slider values you had when you saved — see [Scene profiles](scene-profiles).

## Browse the catalog

Click **Browse effects** and go hunting:

* **Search** by name
* **Tags**, **favorites**, and **filters** to narrow the list
* **Import** community effects you’ve downloaded

Handy keys while the browser is open:

* **Escape** — close
* **Enter** — activate the highlighted effect
* **Arrow keys** — move the highlight (when you’re not typing in search)

## Playback controls

| Control | What it does |
| ------- | ------------ |
| **Prev / Next** | Step through the list |
| **Shuffle** | Surprise me |
| **Auto cycle** | Keep advancing on its own |
| **Pause** | Freeze effect output on hardware. Shows a **play** icon while paused or when **Toggle Lights** turned everything off. |

## Tune an effect

Every effect brings its own controls. Changes stick per effect even before you save a Scene — so you can wander and come back.

**Brightness** in this panel scales all output. Strips too dim on camera? Cranked too hot? Start here.

## Default device layout

Some effects can load a saved layout when you pick them (**Default device layout**). Useful when an effect expects components in specific spots.

## Effect console

The **terminal** button in the **lower-right** of the canvas opens the effect runtime log — errors, `console.log` output, the works. **Escape** to close.

Effect and hardware frame rates show up in the **bottom status bar** if you’re curious.
